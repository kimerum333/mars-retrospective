# MARS 백엔드 API 명세 문서

이 디렉토리는 MARS 프로젝트의 백엔드 API 명세 문서입니다. 
NestJS 기반 백엔드의 모든 엔드포인트를 도메인별로 Markdown 파일로 
정리한 것으로, 프로젝트 진행 중 프론트엔드 팀원들과의 API 연동을 
위해 작성되었습니다.

회고 문서의 배경과 이 문서의 작성 동기에 대해서는 
[메인 README의 담당 영역 1번 섹션](../README.md#1-nestjs-기반-백엔드--거래-시뮬레이션과-포트폴리오-관리)
을 참고해주세요.

## 파일 구성

- `users.md` — 회원 가입, 로그인, 로그아웃, 비밀번호 초기화
- `wallet.md` — 초기 지갑 생성, 잔고 조회, 잔고 변경
- `stock.md` — 종목 검색, 차트 데이터, 상세 정보, 종목 목록
- `trade.md` — 매수/매도 주문 (낙관적 락 패턴 기반 가격 검증 포함)
- `portfolio_mine.md` — 내 포트폴리오 조회, 거래 내역
- `portfolio_like.md` — 관심 종목 관리
- `ai.md` — AI 추천 및 사용자 선호 설정

## 작성 시점

본 문서들은 프로젝트 진행 중(2025년 3~6월) 작성된 원본이며, 
회고 레포에 공유하기 위해 실제 서버 IP 등 민감 정보만 `{server}` 
플레이스홀더로 치환한 것 외에는 수정하지 않았습니다.
